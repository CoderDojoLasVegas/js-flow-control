if

There are 3500 cars on a car lot. On average 20% of cars are Blue. This particular car lot has 563 Blue cars. Write code that will alert the user if the car lot is average, above average, or below average. 

    var carlot = 3500;
    var avgBlue = 0.20;
    var knownBlue = 563;
    if (carlot*avgBlue > knownBlue) {
        alert("The car lot is Below Average.");
    }
    if (carlot*avgBlue == knownBlue) {
        alert("The car lot is Average.");
    }
    if (carlot*avgBlue < knownBlue) {
        alert("The car lot is Above Average.");
    }

if/else if/else

We want to know if an airplane is full of passengers or it has more room. Assume this particular airplane holds 160 passengers, and the number of passengers on the airplane is equal to the number of people in this room. We also want to cancel the flight if there are less than 20% of the seats are occupied. 

    var minload = 0.20;
    var passengers = 4;
    var maxload = 160;
    if (passengers < minload*maxload) {
        alert("The Flight is Cancelled.");
    } else if (passengers >= maxload) {
        alert("The Plane is full.");
    } else {
        alert("The Plane has more room.");
    }
