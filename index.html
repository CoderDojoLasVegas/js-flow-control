JavaScript Flow Controls

What is Flow Control

Programs execute along a particular route or pathway, and that route can be altered based on differing conditions. The process of evaluating conditions and altering the route of the program is called Flow Control.

Examples of flow control in action would be sorting a list, or counting the number of Blue cars in a list of cars. Programs that walk through a list sorting it, or make decisions based on the conditions of items in the list use flow control. There are two basic types of flow controls, Decision types and Loop types.

Decision Types

Decision types change the flow based on a condition. Many statements in JavaScript can evaluate conditions, but the ones we consider decision types are:

if
Makes a decision based on a condition
if/else
if/else As if, but with an alternative clause
switch
Multi-if statement

Loop Types

Statements that continuously evaluate until a particular condition or conditions are met are considered loop types. Loop types include:

for
An iterative loop
for/in
Iterates through the properties of an object
while
Performs code while a condition is true
do/while
Performs code, then checks for the condition being true
break
Stops execution of a loop
continue
Skips remaining iteration of a loop

if

IF is the simplest flow control. It alters the flow of the program according to a condition. The syntax for IF is: 

if (condition) {...code...}

As with all the flow control keywords, we only need {} if we are to execute more than one line of code.

Using if, we can test if a car is Blue. We can also test strings, numbers, and other data types. For example "is x greater than y", for which we would use the greater than symbol (>). This comes in handing when sorting things.

Example: 

    var x = 10;
    var y = 8;
    if (x > y) {
        alert(x);
    }

Now you try writing code for the following comparisons:

  • We want to know if an airplane is full of passengers or it has more room. Assume this particular airplane holds 160 passengers, and the number of passengers on the airplane is equal to the number of people in this room.
  • There are 3500 cars on a car lot. On average 20% of cars are Blue. This particular car lot has 563 Blue cars. Write code that will alert the user if the car lot is average, above average, or below average.

if/else

There is another extension to the IF flow control, which involves handling the alternative case. The syntax for if/else is: 

if (condition) {...code...} else {...code...}

This allows us to specify an alternative action to take when the condition is false.

Using if/else, we can choose to go to the beach or the zoo based on the temperature. If the temperature is above 90°F we want to go to the beach, otherwise we will spend the day at the zoo.

Example: 

    var temperature = 87;
    var beachTripTemp = 90;
    if (temperature > beachTripTemp) {
        alert("Get to the beach NOW!");
    } else {
        alert("Better spend the day at the zoo.");
    }

  • There are 3500 cars on a car lot. On average 20% of cars are Blue. This particular car lot has 563 Blue cars. Write code that will alert the user if the car lot is average, above average, or below average.

if/else if/else

There is another extension to the if/else flow control, which involves handling additional conditions within the alternative case. The syntax for if/else if/else is: 

if (condition) {...code...} else if (condition) {...code...} else {...code...}

This allows us to specify an additional condition to evaluate when the condition is false.

Using if/else if/else, we can choose to go to the beach or the zoo based on the temperature, or stay at home if it's raining. If the temperature is above 90°F we want to go to the beach, otherwise we will spend the day at the zoo, but if it's raining, we'll want to stay home no matter what.

Example: 

    var rain = false;
    var temperature = 87;
    var beachTripTemp = 90;
    if (rain == true) {
        alert("Better stay home.");
    } else if (temperature > beachTripTemp) {
        alert("Get to the beach NOW!");
    } else {
        alert("Spend the day at the zoo.");
    }

Switch

The switch statement selects an option from a variable. The syntax for switch is: 

    switch (variable) {
    case value1 : {...code1...}
    case value2 : {...code2...}
    ...
    case valueN : {...codeN...}
    default: {...default code...}

The variable may be either a string or integer variable. The value of variable is matched against the list of values inside the switch statement, and if a match is found, then the associated code is performed. If no match is found, the code inside the default clause is performed.

One more detail, if, say, value 1 is matched, then unless we do something about it, all the codes will be performed, i.e. code1, code2, ... , codeN and default code will all be executed. To prevent this we add break as the last line in each code clause.

for

The for loop allows you to specify an iterator, a start value, and a stop condition. The syntax for the for loop is: 

    for(startValue; stopCondition; iteration){
        document.write(i);
    }

Example: 

    for(i=0; i<10; i++){
        document.write(i);
    }
